## Start the app on Android Mobile ##

1. Download "Expo Go" App from the app store.
2. cd "REACT-NATIVE-EXERCISEAPP
3. npm install
4. npx expo Start
5. Start "Expo Go" app in your mobile device, scan QR code in the terminal using the QR code scanner inside expo app.
6. App starts.

####################################################

## Start app on webb ##
## The application will look a bit different when opening in the webb because
   its only built for mobile.

1. cd "REACT-NATIVE-EXERCISE
2. npm install
3. npx expo start
4. press the key "w" on your keyboard to start on webb.
